## WellSoft - система для работы с данными со скважин

## Требования для запуска
- Операционная система: Windows (приложение использует WPF).
- Установленный .NET SDK (версия 8.0 или выше) для сборки и запуска из исходников, либо наличие .NET Runtime 8.0, если предоставлен скомпилированный исполняемый файл.
- Visual Studio 2022 (или другая IDE) для разработки и запуска в режиме отладки.

### Установка через консоль
1. Откройте **Tools → NuGet Package Manager → Package Manager Console**.
2. В выпадающем списке "Default project" выберите основной проект `WellSoft`.
3. Выполните команду для установки единственного необходимого пакета:
   ```powershell
   Install-Package Microsoft.Extensions.DependencyInjection
   ```
4. Затем в выпадающем списке выберите тестовый проект `WellSoft.Tests` и выполните:
   ```powershell
   Install-Package xunit
   Install-Package xunit.runner.visualstudio
   Install-Package Microsoft.NET.Test.Sdk
   Install-Package Moq
   Install-Package FluentAssertions
   ```

### Установка через UI
- Щёлкните правой кнопкой на проекте → **Manage NuGet Packages**.
- Перейдите на вкладку **Browse**.
- Найдите и установите указанные выше пакеты для каждого проекта.

## Сборка и запуск

### Способ 1: Запуск через Visual Studio
1. Откройте файл решения `WellSoft.sln` в Visual Studio.
2. Убедитесь, что запускаемым проектом выбран `WellSoft` (WPF-проект).
3. Нажмите клавишу `F5` или выберите **Debug → Start Debugging**.
4. Приложение скомпилируется и запустится.

### Способ 2: Запуск через командную строку (dotnet)
1. Откройте терминал в папке, содержащей файл решения `WellSoft.sln`.
2. Выполните команду:
   ```bash
   dotnet build
   ```
3. Перейдите в папку с выходными файлами WPF-проекта (например, `WellSoftr/bin/Debug/net8.0-windows`).
4. Запустите исполняемый файл `WellSoftr.exe`.

## Дополнительные реализованные пункты

### 1. UI с таблицей скважин и списком интервалов (WPF)
- Разработан графический интерфейс на WPF с использованием паттерна MVVM (без сторонних библиотек, ручная реализация `INotifyPropertyChanged` и `ICommand`).
- Интерфейс содержит три таблицы: сводка по скважинам, интервалы выбранной скважины, список ошибок.
- Реализована навигация: выбор скважины в сводке автоматически обновляет список интервалов.

### 2. Асинхронная загрузка файла с использованием async/await
- Загрузка и обработка CSV-файла выполняется в фоновом потоке (`Task.Run`), чтобы не блокировать пользовательский интерфейс.
- Во время обработки отображается статус «Загрузка...».
- После завершения данные обновляются через `Dispatcher.Invoke`, так как коллекции привязаны к UI.

### 3. Экспорт сводной информации в JSON
- Реализована команда экспорта, доступная только при наличии данных.
- Используется встроенный `System.Text.Json` для сериализации списка объектов `WellData`.
- Имя файла по умолчанию содержит дату и время для предотвращения перезаписи.
- При успешном экспорте выводится сообщение в строке состояния.

### 4. Unit-тесты (xUnit)
- Создан отдельный проект `WellSoft.Tests` с тестами на xUnit.
- Покрыты компоненты:
  - валидация интервалов (`Interval.Validate`)
- Для запуска тестов используйте **Test Explorer** в Visual Studio или команду `dotnet test`.
